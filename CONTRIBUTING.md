# Contributing

歓迎: **プロンプト派生の追加PR** / ドキュメント改善 / バグ報告。

## プロンプトの追加（派生）の方針
- ディレクトリ: `docs/prompts/templates/` に **新規ファイル**として追加してください。
- ファイル名: `chat_to_gh_issue_backup_prompt_<variant>.md`（例: `_claude.md`, `_gpt.md` など）
- 品質基準:
  - **SOP（手順説明）を本文/コメントに混在させない**（SOPはテンプレ/docs側へ）。
  - **[MASKED]** ルール（機微の具体例は `SECURITY.md` を参照）。
  - **長文対策**（≤30KB／≥10秒／1バッチ≤5、失敗時はコメント代替）を明示。
  - **PREPENDの前提**（アンカー直下にGFM半角`|`で1行追加）を明示。
- ライセンス: 既定の **MIT** に同意の上、第三者の権利を侵害しないこと。

## 提案の出し方
- 変更点の要旨（何が改善されるか）をPR本文に1-3行で記述。
- 可能であれば、**ダミーE2E**手順（#2相当）での確認結果を併記。

## 連絡
- 議論が必要な場合は Issue を作成してください（テンプレは最小主義です。導線の簡素さを優先します）。

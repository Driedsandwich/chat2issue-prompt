# SECURITY

本テンプレートは、**情報流出を防ぎつつ、手動（UI/MCP）で安全に運用**することを目的としています。

## マスキング指針
- 機微情報は **`[MASKED]`** に置換してから記録します。
- 機微の例：APIキー／アクセストークン／パスワード／個人名やメール／内部URL・ネットワーク構成／未公開のプロジェクト名 など。
- 本文に**日付は書かない**（GitHubのタイムスタンプを正とします）。

## 禁止事項
- 機微情報の**平文貼り付け**
- 内部URLや固有IDの**不用意な公開**（必要時のみ、正当性を確認して最小限に）
- 本文/コメントに**SOP（手順）や運用説明**を書き込むこと（SOPは `docs/` に集約）

## 安心宣言（テンプレの設計）
- 本テンプレートは**自動的な外部送信・外部通信を行いません**。
- MCPを用いる場合も、**利用者の明示的な操作**でのみ投稿が行われます。

## 脆弱性の報告（任意）
- 連絡先がある場合は、Issue または連絡手段を README に追記してください。

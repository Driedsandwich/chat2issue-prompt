# Prompts

本テンプレートは**プロンプト本文を同梱しない方針**でしたが、
**参考用の正本**として、`docs/prompts/templates/` に例を配置します。

- 目的：利用者が自分のLLM/ユースケースに合わせて**派生**しやすくする
- セキュリティ：機微は `[MASKED]`、SOPは本文/コメントに混在させない（`SECURITY.md` 参照）
- 派生の投稿（PR）は `CONTRIBUTING.md` を参照
